# schedule-v1 — Project Schedule

The Schedule schema will define a standardized format for construction project schedules, including activities, dependencies, milestones, and resource assignments.

**JSON Schema:** [`spec/schemas/schedule-v1.json`](https://github.com/pelles-ai/taco/blob/main/spec/schemas/schedule-v1.json)

**Status:** Placeholder. The schema file exists but the detailed structure is in development.

## Planned Fields

- Project identification and date range
- Activities with start/end dates, durations, and dependencies
- Milestones and phase markers
- Resource assignments by trade
- Critical path identification
- Integration with P6, MS Project, and other scheduling tools
