# change-order-v1 — Change Order

The Change Order schema will define a standardized format for construction change order analysis, including cost impact, schedule impact, and scope modifications.

**JSON Schema:** [`spec/schemas/change-order-v1.json`](https://github.com/pelles-ai/taco/blob/main/spec/schemas/change-order-v1.json)

**Status:** Placeholder. The schema file exists but the detailed structure is in development.

## Planned Fields

- Original scope reference
- Proposed changes with descriptions
- Cost impact (additions and deductions)
- Schedule impact (delays, accelerations)
- Affected trades and CSI divisions
- Approval workflow status

:::tip Contributing
This schema is looking for contributors. If you work with change order management and want to help define the standard, please [open an issue](https://github.com/pelles-ai/taco/issues) describing your use case.
:::
